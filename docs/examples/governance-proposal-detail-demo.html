Register new ERC721 auction factory for generative art collections

Submitted 2 days ago 0x742d...35c9

Description

This proposal registers a new ERC721 auction factory contract that enables artists to launch generative art collections with Dutch auction mechanics. The factory includes embedded royalty distribution to support sustainable creator economies.

Key features include:

  • Dutch auction mechanics with configurable starting price and decay rate
  • Embedded royalty splits between creator and protocol
  • Support for on-chain generative rendering
  • Integration with existing vault alignment system

The factory contract has been audited by Trail of Bits and is ready for deployment. Registration will enable artists to create sustainable generative art projects with built-in revenue sharing.
