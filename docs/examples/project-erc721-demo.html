About This Project

Infinite Portraits is an ongoing generative art series exploring the boundary between human likeness and algorithmic abstraction. Each piece is created using custom software that interprets portrait photography through layers of computational distortion.

The artist queues pieces as they are created, with each one backed by an ETH deposit that sets the minimum bid and demonstrates commitment to the work. Pieces are auctioned across 3 parallel lines, allowing multiple collectors to compete simultaneously. 80% of proceeds go to the creator, while 20% flows to the WETH alignment vault.

How It Works
  1. Creator queues a piece with ETH deposit (= minimum bid)
  2. Piece is assigned to a line via round-robin (1, 2, 3, 1, 2, 3...)
  3. If the line is idle, the auction starts immediately
  4. Bidders compete in an English auction for the duration
  5. Late bids (within 10 minutes of end) extend the auction
  6. Winner receives the NFT, creator gets deposit + 80% of bid
  7. 20% of the winning bid goes to the alignment vault
If a Piece Goes Unsold

If the auction ends with zero bids, the NFT is never minted and the creator's deposit is forfeited to the protocol treasury. This mechanism ensures creators have genuine confidence in each piece they queue.
